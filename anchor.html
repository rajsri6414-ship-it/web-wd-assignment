HTML Anchor Tag

The HTML <a> (anchor) tag is used to create hyperlinks, which allow users to click from one page to another or to different sections within the same page. The href attribute specifies the destination of the link.

Example of an Anchor Tag:

Click here to visit Example.com.

Anchor Tag with Internal Link:

Go to the Section 1 below.

Section 1

This is Section 1 of the document. You can return to the top by clicking here.
